
Le code source java permettant de lancer la bataille navale se trouve dans le répertoire src/ 

Pour lancer le jeu en mode console les commandes principales sont :

- "sh compilescripts/compile.sh" pour compiler le programme et créer un répertoire bin contenant les .class.
- "sh compilescripts/runConsole.sh " pour lancer le programme.

Pour lancer le jeu  un mode MVC il vous suffit de lancer BatailleNavale.jar
